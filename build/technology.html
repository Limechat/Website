What is a Level 3 Chatbot?

Level 3 chatbots can understand context.

User Profile

This consists of the user's preferences, personal details, chat history, and purchasing history

User Goal

This comprises of the specific task the user wants completed and its supporting details

Conversation History

This consists of the user's preferences, personal details, chat history, and purchasing history

World Knowledge

Based on the user profile and goal, words can have different interpretations

How Level 3 is better than Level 2

Its a game changer for D2C brands.

1

Gracefully Handle Any Goal

The assistant will identify and smartly complete user goals. For highly complex queries it will handle seem-less human handoff

2

Disambiguation

Human languages usually contain a lot of ambiguity between words. It is important to remove uncertainty using contextual features.

3

Change of Context

The user might at any point in time change their mind. This can affect any type of contextual data or lead to a change of user goal

4

Extract Contextual Data

Language can contain a lot of relevant structured data (e.g. dates, cities) that needs to be extracted from unstructured text.

5

Read/Write Contextual Data

The assistant should maintain contextual data in a database via APIs, to avoid restarting user chats from scratch every time.

6

Business Logic

Companies have personalized systems to integrate with, unique products to sell, and specific rules with which to address the user
